# Do I Like This Song

A lightweight Electron desktop app for macOS.

Features
- Open a music folder
- Browse tracks in that folder
- View title, artist, album, and artwork
- Play music and skip forward/back 10 seconds
- Give thumbs up or down for each track
- Moves the liked and disliked tracks into new subdirectories so you know which ones you want

Run
1. npm install
2. npm start
